## Instructions to Run the Project

### Prerequisites
- Node.js (version 14 or higher)
- npm or yarn

### Setup

1. **Clone the Repository**:
   ```bash
   git clone <url>
   ```

2. **Navigate to the Project Directory**:
   ```bash
   cd roadmap/8-mini-project
   ```

### Backend Setup

Setup

3. **Navigate to the Backend Directory**:
   ```bash
   cd backend
   ```

4. **Install Backend Dependencies**:
   ```bash
   npm install
   ```

5. **Compile TypeScript Code**:
   ```bash
   npx tsc
   ```

6. **Run the Backend**:
   ```bash
   npm run dev
   ```

### Access the Backend
- Backend runs on `http://localhost:3000` 

### Frontend Setup

6. **Navigate to the Frontend Directory**:
   ```bash
   cd ../frontend
   ```

7. **Install Frontend Dependencies**:
   ```bash
   npm install
   ```

8. **Run the Frontend**:
   ```bash
   npm run dev
   ```

### Access the Application
- Open your web browser and go to `http://localhost:5173` (or the appropriate port for your Vite app).
